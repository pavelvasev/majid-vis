# vis2

Show files of format:
```
{
"lines": [
   [x1,y1,z1,x2,y2,z2,c1,c2,c3], 
   ..repeated.. 
 ],
"spheres": [
   [x1,y1,z1,r,c1,c2,c3],
    ..repeated.. 
 ],
"text": [
   [x1,y1,z1,"title"], 
   ..repeated.. 
 ]
}
```
where c_i is in range 0..1
